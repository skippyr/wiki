tdk_getWindowDimensions Function

Declaration

#include <tdk.h>

int tdk_getWindowDimensions(struct tdk_Dimensions* dimensions);

About

Gets the total of columns and rows present in the terminal window.

Parameters

  • dimensions: a pointer to a tdk_Dimensions struct that receives the terminal window dimensions.

Return Value

If the function succeeds, the return value is 0.

If the function fails, the return value is -1. It fails if:

  • On Windows, the standard output and error streams are being redirected or piped.
  • On Linux, all the standard streams are being redirected or piped.

Example

#include <tdk.h>

static struct tdk_Dimensions g_windowDimensions;

int main(void)
{
    if (tdk_getWindowDimensions(&g_windowDimensions))
    {
        tdk_writeErrorLine("main: could not get the terminal window dimensions.");
        return 1;
    }
    tdk_writeLine(":: Total of columns: %hu.", g_windowDimensions.totalOfColumns);
    tdk_writeLine(":: Total of rows   : %hu.", g_windowDimensions.totalOfRows);
    return 0;
}
